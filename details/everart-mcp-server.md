## Overview

EverArt was a reference MCP server that demonstrated how AI image generation models can be integrated with LLMs through the Model Context Protocol. It has been archived and moved to the servers-archived repository.

## Features

- AI image generation using various models
- Demonstrates multimodal MCP capabilities
- Reference implementation for creative tool integration

## Status

This server is now archived and can be found in the servers-archived branch of the modelcontextprotocol/servers repository. It is maintained as a reference example rather than an actively developed server.

## Purpose

As part of the official reference servers collection, EverArt demonstrated how MCP can be used to give Large Language Models access to image generation capabilities.

## Pricing

Free and open-source reference implementation.
