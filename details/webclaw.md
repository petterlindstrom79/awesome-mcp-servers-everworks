## Overview

Webclaw is a web content extraction tool designed for LLM pipelines. It delivers clean markdown or structured JSON from any URL using browser-grade TLS fingerprinting, eliminating the need for a headless browser.

## Features

- Clean markdown extraction from any URL
- Structured JSON output for programmatic consumption
- Browser-grade TLS fingerprinting for reliable access
- No headless browser required
- CLI interface for command-line usage
- REST API for programmatic integration
- MCP server interface for LLM agent integration

## Interfaces

- CLI (Command Line Interface)
- REST API
- MCP Server
