## Overview

Calendly Scheduling MCP is a Model Context Protocol server that integrates with Calendly's scheduling platform. It enables AI assistants to schedule meetings, manage appointments, and handle booking workflows efficiently through natural language conversations.

## Features

- **Meeting Scheduling**: Create and manage Calendly booking links
- **Availability Management**: Query and update availability settings
- **Event Type Management**: Configure different types of meetings
- **Invitee Management**: Handle meeting invitations and RSVPs
- **Calendar Integration**: Sync with connected calendars
- **Booking Page Customization**: Configure booking page settings
- **Time Zone Handling**: Automatic time zone conversion for invitees
- **Buffer Time Configuration**: Set buffer time between meetings

## Scheduling Features

- Generate personalized booking links
- Share availability through AI conversations
- Schedule one-on-one and group meetings
- Set up round-robin scheduling for teams
- Configure collective scheduling for team availability
- Manage recurring meeting slots
- Handle last-minute cancellations and rescheduling

## Use Cases

- Schedule client meetings through AI assistants
- Manage consultant and advisor bookings
- Coordinate interview scheduling
- Set up sales calls and demos
- Handle customer support appointments
- Organize team meetings and sync-ups
- Automate meeting logistics

## Integration Capabilities

- Connect with Google Calendar, Outlook, and Office 365
- Integrate with video conferencing platforms (Zoom, Google Meet, Microsoft Teams)
- Sync with CRM systems
- Webhook support for custom workflows
