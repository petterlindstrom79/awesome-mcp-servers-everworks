# MemoryLens MCP

On-demand .NET memory profiling MCP server wrapping JetBrains dotMemory. Uses heuristic rule engine with 10 rules to detect memory leaks, LOH fragmentation, anti-patterns, and provides AI-actionable code fix suggestions for Windows.
