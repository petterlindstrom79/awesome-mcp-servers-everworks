## Overview

MindsDB Universal Database MCP is a unified Model Context Protocol server that connects AI models directly to database ecosystems. It supports cloud-based, on-premises, and hybrid deployments across major database platforms including Oracle, PostgreSQL, MongoDB, MySQL, Snowflake, and more.

## Supported Databases

- **Relational**: PostgreSQL, MySQL, Oracle, SQL Server, MariaDB
- **Cloud Data Warehouses**: Snowflake, BigQuery, Redshift
- **NoSQL**: MongoDB, Cassandra, CouchDB
- **Time Series**: InfluxDB, TimescaleDB
- **Graph**: Neo4j
- **In-Memory**: Redis
- **And 100+ more data sources**

## Features

- **Universal Connector**: Single interface for multiple database types
- **Natural Language Queries**: Convert conversational queries to database-specific syntax
- **AI-Powered Analytics**: Built-in machine learning capabilities

- **Cross-Database Queries**: Join data across different database systems
- **Secure Connections**: Enterprise-grade security
