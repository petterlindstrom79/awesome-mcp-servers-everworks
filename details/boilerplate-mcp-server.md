## Overview

TypeScript MCP server boilerplate that provides IP lookup tools/resources, CLI support, and an extensible structure for connecting AI systems to external data sources like ip-api.com. Ideal template for creating new MCP integrations via Node.js.

## Features

- IP lookup tool (ip-api.com integration)
- CLI support
- Extensible project structure
- TypeScript/Node.js based
- Ideal template for new MCP integrations
