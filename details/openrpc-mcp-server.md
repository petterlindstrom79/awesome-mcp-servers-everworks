## Overview

OpenRPC MCP Server bridges the Model Context Protocol with OpenRPC specifications, allowing MCP clients to discover and interact with JSON-RPC services through standardized interfaces.

## Features

- Exposes JSON-RPC functionality through MCP
- Leverages OpenRPC specifications for service discovery
- Standardized protocol bridge between MCP and JSON-RPC
- Automatic tool generation from RPC definitions
- Supports typed schema validation
