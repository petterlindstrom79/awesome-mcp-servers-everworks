# MCP Servers

A collection of servers for the Model Context Protocol. It curates top MCP servers including sponsors like Bright Data, Scout Monitoring, Alpha Vantage, and official ones like Anki MCP, Browserbase, Chrome DevTools MCP, Cloudflare, E2B, Firecrawl, GitHub, Google, and many more for AI assistants and agents. Relevant for discovering MCP tools, libraries, and integrations in cloud-native, development, data access, and AI workflows.
