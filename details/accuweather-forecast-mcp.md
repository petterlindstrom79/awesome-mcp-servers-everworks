## Overview

AccuWeather Forecast MCP is a Model Context Protocol server that provides hourly and daily weather forecasts using the AccuWeather API. It enables AI assistants to access accurate, localized weather data and deliver forecasts through natural language conversations.

## Features

- **Hourly Forecasts**: Get detailed hour-by-hour weather predictions
- **Daily Forecasts**: Access multi-day weather forecasts
- **Location Search**: Find weather data by city name, ZIP code, or coordinates
- **Current Conditions**: Real-time weather observations
- **Temperature Data**: Current, feels-like, and forecasted temperatures
- **Precipitation Info**: Rain, snow, and precipitation probability
- **Wind Information**: Wind speed, direction, and gusts
- **Weather Alerts**: Severe weather warnings and advisories

## Weather Data Provided

- Temperature (current and forecast)
- Humidity levels
- Precipitation probability and amounts
- Wind speed and direction
- UV index
- Visibility
- Cloud cover
- Atmospheric pressure
- Sunrise/sunset times

## Use Cases

- Query weather conditions through AI conversations
- Plan activities based on weather forecasts
- Get weather alerts for travel planning
- Integrate weather data into scheduling applications
- Build weather-aware automation workflows
- Create personalized weather briefings

## API Requirements

Requires an AccuWeather API key:
- Free tier available with limited requests per day
- Supports location search and basic forecasts

## Pricing

Free and open-source server. AccuWeather API offers free tier with daily limits.
