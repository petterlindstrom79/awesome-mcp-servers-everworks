## Overview

Agentset AI MCP Server provides multi-agent capabilities via MCP for RAG on private data.

## Features

- Multi-agent tool calling and chaining
- Model switching support
- Stateful context persistence

## Use Cases

- Agent workflows for data analysis
- Task delegation in research pipelines

## Comparisons

Native MCP protocol enables direct LLM integration vs LangChain's higher-level abstractions.
