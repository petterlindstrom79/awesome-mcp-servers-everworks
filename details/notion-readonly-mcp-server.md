## Overview

The Notion ReadOnly MCP Server is a variant of the Notion MCP Server that provides only read-only access to Notion resources. It includes tools such as API-retrieve-a-database for querying Notion databases without modification capabilities.

## Features

- Retrieve databases from Notion workspaces
- Read-only access ensures data safety and security
- Compatible with any MCP client
- Ideal for analytics, search, and query-based workflows
