## Overview

AWS Terraform Best Practices MCP is a specialized Model Context Protocol server developed by AWS Labs that provides prescriptive Terraform advice specifically for AWS deployments. It includes infrastructure as code patterns, security compliance checking with Checkov, and AWS-specific best practices guidance.

## Features

- **AWS Best Practices**: Prescriptive guidance for AWS infrastructure
- **Security Compliance**: Integrated Checkov security scanning
- **Infrastructure Patterns**: Proven AWS architecture patterns
- **Cost Optimization**: Recommendations for cost-effective configurations
- **Well-Architected Framework**: Alignment with AWS Well-Architected
- **Service Integration**: Optimal configuration for AWS services
- **Compliance Checking**: CIS, PCI-DSS, HIPAA compliance validation
- **Automated Remediation**: Suggestions for security issue fixes
