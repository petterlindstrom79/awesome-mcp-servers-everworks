## Overview

mcphost is a command-line host application that bridges Large Language Models with external tools via the Model Context Protocol. It serves as an MCP client that enables LLMs to discover and use tools, access resources, and leverage prompts from MCP servers.

## Features

- Connects LLMs to external MCP servers for tool access
- Enables tool discovery and execution through standardized MCP protocol
- CLI-based interface for seamless workflow integration
- Supports multiple MCP server connections
- Allows LLMs to interact with diverse tools and data sources
