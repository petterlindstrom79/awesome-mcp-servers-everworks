## Overview

SimpleLocalize's official MCP server extension allows AI agents like ChatGPT, Claude, and other MCP-compatible clients to access translation keys and manage localization workflows. It provides a seamless bridge between AI assistants and the SimpleLocalize translation management platform.

## Features

- **Translation Key Access**: Retrieve and use translation keys in AI conversations
- **Multi-agent Support**: Works with ChatGPT, Claude, and other AI assistants
- **Online i18n Editor**: Web-based translation editor for developers
- **Real-time Updates**: Changes sync immediately across all connected clients
- **Version Control**: Track translation changes and history
- **Collaboration**: Enable team members to work on translations together
- **Project Management**: Organize translations by project and namespace

## Use Cases

- Query translation keys during development conversations
- Update translations through natural language commands
- Review translation coverage with AI assistance
- Generate translation keys from source code
- Maintain consistency across multilingual projects
