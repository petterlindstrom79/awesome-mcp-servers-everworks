## Overview

AWS CloudGuard MCP is an automated incident response system for AWS accounts. It analyzes CloudWatch logs and alarms, diagnoses issues, and creates JIRA tickets — all orchestrated through Claude's Model Context Protocol.

## Features

- Monitor AWS CloudWatch logs and alarms for given services
- Automated incident diagnosis using AI analysis
- Automatic JIRA ticket creation for identified incidents
- Integration with Claude MCP for intelligent decision making
- Covers full incident lifecycle from detection to documentation
