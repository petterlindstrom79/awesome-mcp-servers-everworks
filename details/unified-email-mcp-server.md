## Overview

Unified Email MCP Server provides a comprehensive email access solution across Gmail, Outlook, iCloud, and generic IMAP providers. It features OAuth2 authentication through browser-based OAuth flows for Gmail and Outlook with automatic token refresh, ensuring seamless and secure email integration.

## Features

- **Multi-Provider Support**: Gmail (REST API), Outlook (Microsoft Graph), iCloud (IMAP), and generic IMAP/SMTP
- **OAuth2 Authentication**: Secure browser-based authentication for Gmail and Outlook
- **Automatic Token Refresh**: Maintains active sessions without manual intervention
- **Unified Interface**: Consistent API across all email providers
- **Full Email Operations**: Read, send, search, and manage emails
- **Attachment Handling**: Download and send file attachments
- **Folder Management**: Organize emails across folders and labels
- **Advanced Search**: Powerful search capabilities across providers

## Email Operations

- Read emails and threads
- Send new emails
- Reply to messages
- Forward emails
- Search inbox and folders
- Download attachments
- Manage folders and labels
- Mark as read/unread
- Archive and delete messages
- Manage drafts

## Authentication Methods

- **Gmail**: OAuth2 with Google authentication
- **Outlook**: OAuth2 with Microsoft authentication
- **iCloud**: App-specific passwords
- **IMAP**: Username/password or OAuth2
