## Overview

mcp-runtime is a pre-configured runtime environment designed to simplify the process of running Model Context Protocol (MCP) servers on Windows. It integrates Node.js (including npx and npm), Python (including uv, uvx, and pip), and Git, allowing developers to focus on building MCP applications like Chatbot clients instead of wrestling with environment configuration.

## Features

- Integrated Node.js with npx and npm
- Integrated Python with uv, uvx, and pip
- Pre-installed Git
- Windows-focused setup
- Eliminates manual environment configuration
