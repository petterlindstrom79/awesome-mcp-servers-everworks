## Overview

SuperGateway is an AI gateway that bridges different MCP transport layers. It can run stdio-based MCP servers over SSE (Server-Sent Events) and expose SSE servers over stdio, enabling flexible deployment patterns for MCP infrastructure.

## Features

- Run MCP stdio servers over SSE transport
- Expose SSE servers as stdio servers
- Flexible transport bridging
- Enables remote MCP server deployment
- Supports AI gateway patterns
