## Overview

Google's remote MCP server for AlloyDB PostgreSQL, a managed service for Postgres databases.

## Features

- PostgreSQL-compatible with columnar engine
- Serverless auto-scaling and high availability
- IAM authentication
- Full SQL query and management operations
- GA remote endpoint

## Use Cases

- AI agent querying of production databases
- Automated DB ops and analytics

## Pricing

Pay-as-you-go for compute, storage, and network.
