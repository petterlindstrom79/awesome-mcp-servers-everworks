## Overview

Helm Chart Analyzer MCP is a specialized Model Context Protocol server that enables AI agents to read and analyze Helm chart dependencies. It provides deep insights into chart structures, dependencies, and potential configuration issues.

## Features

- **Dependency Analysis**: Automatically parse and understand Helm chart dependency trees
- **Configuration Review**: Analyze values files and identify potential misconfigurations
- **Version Compatibility**: Check dependency version compatibility and conflicts
- **Security Scanning**: Identify known vulnerabilities in chart dependencies
- **Best Practices**: Suggest improvements based on Helm best practices
- **Documentation Generation**: Auto-generate documentation from chart metadata
- Repository authentication and secure access

## Installation

Available as a container image:

```bash
Docker pull ghcr.io/v3ro/helm-chart-mcp:0.0.1
```

## Use Cases

- Understand complex Helm chart dependency structures
- Audit Helm charts for security and best practice compliance
- Troubleshoot deployment issues related to dependencies
