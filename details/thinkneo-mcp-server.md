## Overview

ThinkNEO MCP Server provides an enterprise AI governance layer that enables organizations to monitor and control their AI usage across multiple providers. It connects Claude, ChatGPT, and any MCP-compatible client to ThinkNEO's centralized control plane.

## Features

- Spend tracking across AI providers
- Runtime guardrails for AI interactions
- Policy enforcement for organizational compliance
- Budget limits and cost controls
- SOC2/GDPR/HIPAA compliance monitoring
- Real-time AI provider health monitoring
- Integration with Claude, ChatGPT, and any MCP client
