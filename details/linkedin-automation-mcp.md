## Overview

LinkedIn Automation MCP is a Model Context Protocol server that enables automated profile browsing, searching, and post interactions on LinkedIn using Playwright for browser automation. It provides secure session management for reliable LinkedIn operations through AI assistants.

## Features

- **Profile Automation**: Browse and analyze LinkedIn profiles
- **Search Capabilities**: Advanced LinkedIn search automation
- **Post Interactions**: Like, comment, and share posts automatically
- **Connection Management**: Send and manage connection requests
- **Messaging**: Automate LinkedIn messaging workflows
- **Browser Automation**: Powered by Playwright for reliability
- **Session Management**: Secure authentication and session handling
- **Rate Limiting**: Built-in protection against LinkedIn limits
