## Overview

The Azure AI Foundry MCP Server provides developers with comprehensive access to Azure's AI ecosystem, including model catalogs, deployment management, knowledge indexing with Azure AI Search, and evaluation tools. This experimental server bridges the gap between AI development and Azure's powerful AI infrastructure.

## Key Capabilities

### Model Discovery & Deployment
- Explore the full Azure AI Foundry model catalog
- Get detailed model information with code samples
- Deploy models to Azure AI Services
- Query quota information and regional availability
- Find optimized models for specific tasks (reasoning, generation, etc.)

### Knowledge Management
- Create and manage Azure AI Search indexes
- Add documents and configure indexers
- Build sophisticated Retrieval-Augmented Generation (RAG) systems
- Configure proper schemas for documentation and knowledge bases

### AI Agent Integration
- Connect with Azure AI Agents
- Query existing agents and their configurations
- Evaluate agent performance in production scenarios

### Evaluation Framework
- Run comprehensive text and agent evaluations
- Generate Markdown reports from evaluation results
- Implement quality assurance and safety checks for AI applications
- Test against quality metrics

### Prototyping Tools
- Get setup instructions for GitHub-based prototyping
- Access Azure AI Foundry Labs for cutting-edge research models
- Generate prototyping code with authentication for immediate testing

## Use Cases

- Deploy a Phi-4 model to Azure AI Services for production use
- Create a new search index for a documentation-based RAG system
- Evaluate an AI agent's responses against quality metrics
- Find the best reasoning model for complex analysis tasks
- Build end-to-end customer support agent workflows

## Experimental Status

This MCP server is experimental and under active development. Features and APIs may change. It is ideal for exploring Azure AI capabilities and building prototypes but should be validated for stability requirements before production use.

## Pricing

Uses Azure AI Foundry pricing — costs depend on model deployments, AI Search usage, and evaluation volume.
