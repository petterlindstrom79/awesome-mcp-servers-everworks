## Overview

Simple MCP integration for Obsidian vaults, optimized for local knowledge retrieval.

## Features

- Search notes and backlinks
- Update and manage Markdown files
- Graph-based navigation

## Use Cases

- Local agent knowledge bases
- RAG over personal notes
- Knowledge graph querying

## Comparisons

Vs vector DBs: Leverages Obsidian's native links and structure for precise, editable docs.

## Pricing

Free and open-source.
