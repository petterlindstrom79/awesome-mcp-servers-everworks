## Overview

ContrastAPI is a security intelligence MCP server that provides comprehensive vulnerability research and threat intelligence capabilities through 20 specialized tools. It operates freely without requiring an API key.

## Features

- 20 security-focused MCP tools
- CVE lookup with EPSS (Exploit Prediction Scoring System) scores
- KEV (Known Exploited Vulnerabilities) catalog integration
- Domain reconnaissance capabilities
- SSL/TLS certificate analysis
- IP reputation checking
- IOC (Indicators of Compromise) lookup
- Exploit database search
- Code security analysis

## Pricing

Free to use — no API key required.

## Use Cases

- Vulnerability assessment and research
- Threat intelligence gathering
- Domain and IP investigation
- Security incident response with IOC lookups
- Code security auditing
