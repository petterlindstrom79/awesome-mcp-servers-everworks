## Overview

A TypeScript MCP server that integrates with CometAPI's API to make users able to generate media content with multiple AI models directly from Claude or any other MCP-compatible apps.

## Features

- Midjourney image generation
- Flux image generation
- Kling video generation
- LumaLabs video generation
- Udio music generation
- Chirp text-to-speech

## Use Cases

- AI-generated content creation
- Media generation within AI chat workflows
- Multi-modal content generation
