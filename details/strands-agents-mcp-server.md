## Overview

Provides Strands Agents SDK documentation via MCP for GenAI tool integration.

## Features

- Multi-agent persistent state management
- Async tool calling and task delegation
- Context messaging for real-time updates

## Use Cases

- Agent workflows for ML training and data processing
- Cross-session task orchestration

## Comparisons

MCP-native async execution superior to LangChain for long-running agents.
