## Overview

AgentPMT enables AI agents to securely make real-world payments using USDC stablecoin on Base blockchain. The platform provides granular budget controls, instant settlement, and API/MCP integration for autonomous transactions.

## Features

- USDC stablecoin payments on Base blockchain
- Granular budget controls for AI agents
- Instant settlement capabilities
- API and MCP integration for autonomous transactions
- Enterprise-grade security with complete spending oversight
- Designed for businesses leveraging AI automation

## Use Cases

- Autonomous AI agent payments
- Business process automation with real-world transactions
- Bridging AI and digital economy operations
