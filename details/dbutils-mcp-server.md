# DButils MCP Server

DButils is an all-in-one MCP service that enables your AI to do data analysis by harnessing versatile types of database (SQLite, MySQL, Postgres, and more) within a unified configuration of connections in a secured way (like SSL). Provides a universal database interface for MCP clients.
