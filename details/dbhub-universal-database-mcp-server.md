## Overview

Universal database MCP server connecting to MySQL, PostgreSQL, SQL Server, SQLite, and more. By Bytebase, this server provides a single interface for multi-database operations.

## Features

- Multi-database support (MySQL, PostgreSQL, SQL Server, SQLite)
- Unified connection management
- Schema exploration
- SQL query execution
- Standardized MCP tools
- Bytebase integration

## Pricing

Bytebase offers open-source and enterprise editions.
