## Overview

YouTube Transcription MCP uses yt-dlp to download subtitles and transcripts from YouTube videos. It connects to claude.ai via Model Context Protocol, allowing you to ask Claude to "Summarize the YouTube video <>" and get instant insights from video content.

## Features

- **Subtitle Download**: Extract subtitles from any YouTube video
- **Transcript Extraction**: Get full video transcripts in plain text
- **Multi-language Support**: Download subtitles in various languages
- **Auto-generated Captions**: Access YouTube's auto-generated captions
- **yt-dlp Integration**: Powered by robust yt-dlp library
- **Batch Processing**: Download transcripts from multiple videos
- **Format Options**: Export in various text formats
- **Timestamp Preservation**: Maintain video timestamps in transcripts

## Use Cases

- Summarize YouTube lectures and tutorials
- Extract key points from conference talks
- Create text versions of video content
- Search within video content using text
- Generate study notes from educational videos
- Analyze video content for research
- Translate video content via transcript
- Create searchable video libraries

## Supported Features

- Download closed captions (CC)
- Extract auto-generated subtitles
- Multiple language subtitles
- SRT, VTT, and plain text formats
- Playlist support
- Live stream transcript capture

## Integration

Works seamlessly with:
- Claude Desktop
- Claude Code
- Other MCP-compatible AI assistants

## Requirements

- yt-dlp installed on system
- Internet connection for YouTube access
