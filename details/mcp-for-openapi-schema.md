## Overview

MCP for OpenAPI Schema automatically parses OpenAPI specifications and exposes them as MCP tools. This allows AI assistants to understand and interact with any API that follows the OpenAPI specification without needing custom MCP servers.

## Features

- Automatic OpenAPI schema parsing
- Generates MCP tools from API definitions
- Works with any OpenAPI-compatible API
- Eliminates need for custom API integrations
- Supports OpenAPI 3.x specifications
