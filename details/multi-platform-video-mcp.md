## Overview

Multi-Platform Video MCP is a universal MCP server that provides capabilities for downloading video transcripts in plain text and videos from various platforms including YouTube, Vimeo, Twitter/X, and TikTok. It enables AI assistants to analyze video content across multiple social media and video platforms.

## Features

- **Multi-Platform Support**: Works with YouTube, Vimeo, Twitter/X, TikTok, and more
- **Transcript Download**: Extract text transcripts from videos
- **Video Download**: Download video files for processing
- **Plain Text Output**: Clean, readable transcript format
- **Batch Processing**: Handle multiple videos simultaneously
- **Format Flexibility**: Support for various output formats
- **Metadata Extraction**: Capture video metadata and details
- **URL Parsing**: Automatically detect platform from URL

## Supported Platforms

- **YouTube**: Full video and transcript support
- **Vimeo**: Professional video platform integration
- **Twitter/X**: Video tweets and replies
- **TikTok**: Short-form video content
- **And more**: Extensible to additional platforms

## Transcript Features

- Download closed captions when available
- Extract auto-generated subtitles
- Convert to plain text format
- Preserve timestamps (optional)
- Multi-language subtitle support
- Clean formatting for AI consumption

## Video Download Capabilities

- Multiple quality options
- Format selection (MP4, WebM, etc.)
- Audio-only downloads
- Thumbnail extraction
- Metadata preservation
