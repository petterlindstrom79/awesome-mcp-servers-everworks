## Overview

The SQL Server MCP Server provides conversational access to SQL Server databases (on-premises, Azure SQL, or Fabric) through natural language. Users set up a connection string and can then query their databases without writing SQL syntax.

## Key Capabilities

- **Natural Language Queries**: Describe what data you need in plain English and get formatted results
- **Schema Management**: Explore and manage database schemas through conversation
- **Data Manipulation**: Perform CRUD operations without writing SQL statements
- **Connection Flexibility**: Works with on-premises SQL Server, Azure SQL, and Microsoft Fabric
- **Context Preservation**: Maintains database connection context across queries

## Use Cases

- Find unfulfilled orders from the last 30 days
- Query database connection details
- Explore database schema and table structures
- Generate and execute complex SQL queries naturally
- Format and analyze query results

## Setup

Configure with a database connection string in your MCP settings. Works with VS Code, Claude Desktop, and other MCP-compatible clients. See the official documentation for complete setup instructions and configuration examples.
