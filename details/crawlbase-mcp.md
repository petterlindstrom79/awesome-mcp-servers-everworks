## Overview

Crawlbase MCP is a Model Context Protocol server that bridges AI agents and LLMs with real-time web data.

## Features

- Web scraping capabilities
- JavaScript rendering support
- Anti-bot protection
- Integration with Claude, Cursor, and Windsurf
- Real-time web data extraction

## Use Cases

- Web content extraction for AI workflows
- Scraping JavaScript-heavy websites
- Real-time data gathering for AI agents
