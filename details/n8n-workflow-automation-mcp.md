## Overview

n8n Workflow Automation MCP is a Model Context Protocol server that brings n8n's powerful visual workflow automation capabilities to AI assistants. It enables building MCP servers and clients using n8n's 400+ integration nodes, allowing complex automation workflows to be created and managed through natural language.

## Features

- **400+ Integrations**: Pre-built nodes for popular services
- **Visual Workflow Design**: Flow-based programming interface
- **Custom Logic**: JavaScript and Python code execution
- **Error Handling**: Built-in retry and error workflows
- **Scheduling**: Cron-based workflow triggers
- **Webhooks**: HTTP endpoints for external triggers
- **Data Transformation**: Built-in data manipulation tools
- **Version Control**: Workflow versioning and Git sync
