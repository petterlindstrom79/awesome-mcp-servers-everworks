## Overview

Vimeo API MCP Server is the official Model Context Protocol server from Vimeo that provides tools for interacting with the Vimeo API. It allows AI assistants to manage videos, transcripts, and retrieve video statistics through a standardized interface, enabling comprehensive video management workflows.

## Features

- **Video Management**: Upload, update, and delete videos
- **Transcript Handling**: Manage video transcripts and captions
- **Statistics Retrieval**: Access detailed video analytics and metrics
- **Metadata Management**: Update video titles, descriptions, and tags
- **Privacy Settings**: Configure video privacy and embedding options
- **Thumbnail Management**: Upload and manage custom thumbnails
- **Collection Management**: Organize videos into showcases and albums
- **Team Collaboration**: Manage team access and permissions

## Video Operations

- Upload videos programmatically
- Update video metadata
- Manage video privacy settings
- Add and edit video descriptions
- Configure player settings
- Manage video collections
- Retrieve video details
- Delete videos

## Transcript Features

- Upload custom transcripts
- Generate automatic captions
- Edit existing transcripts
- Multi-language subtitle support
- SRT and VTT format support
- Transcript search and retrieval

## Analytics and Statistics

- Geographic distribution
- Referral sources

## Use Cases

## Integration

- Claude Desktop
- Claude Code
- Cursor
- Other MCP-compatible clients
