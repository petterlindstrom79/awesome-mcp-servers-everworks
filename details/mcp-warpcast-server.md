## Overview

MCP Warpcast Server integrates the Farcaster social protocol with AI assistants via the Model Context Protocol. It enables LLMs to read, create, and manage social content on Warpcast/Farcaster.

## Features

- Read Farcaster casts and feeds
- Post new casts through MCP
- User profile queries
- Interaction with decentralized social protocol
- Farcaster network engagement
