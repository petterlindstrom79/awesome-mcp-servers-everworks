# rcon-mcp

rcon-mcp is an MCP server for Dockerized Minecraft instances using itzg's minecraft-server container, enabling AI agents to administer servers via RCON with security through protocol isolation and no persistent shell access. It exposes comprehensive RCON command tools for LLMs, supporting programmatic execution without custom scripting. Use cases include AI-orchestrated in-game builds, real-time server moderation, player management, and automated backups; unlike basic RCON clients, it's optimized for AI workflows with direct MCP integration for tools like Claude Desktop, Cursor, and Zed.
