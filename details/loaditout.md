## Overview

Loaditout is a universal registry and discovery platform for MCP servers and agent skills, cataloging over 20,000 tools with security grading and easy installation.

## Features

- Registry of over 20,000 MCP servers and agent skills
- A/B/C/F security grading system for each tool
- CLI installer: `npx loaditout add`
- Verified publisher badges for trusted sources
- Curated packs for common use cases
- Security assessment for all listed tools

## Security Grading

- **A**: Thoroughly vetted, high security standards
- **B**: Good security practices, minor improvements possible
- **C**: Basic security, use with caution
- **F**: Known security concerns, not recommended

## Use Cases

- Discovering new MCP servers for agent workflows
- Evaluating security posture of MCP tools before installation
- One-command installation via CLI
- Finding curated tool collections for specific domains
