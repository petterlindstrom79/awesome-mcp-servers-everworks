## Overview

X (Twitter) Data MCP provides comprehensive access to Twitter/X data including profiles, tweets, trends, and communities via various APIs. It enables social listening, content analysis, and audience engagement through AI interfaces with full Twitter ecosystem integration.

## Features

- **Tweet Management**: Post, read, and delete tweets
- **Profile Access**: View and analyze user profiles
- **Timeline Operations**: Access and filter timelines
- **Trend Monitoring**: Track trending topics and hashtags
- **Community Features**: Interact with Twitter Communities
- **Search Capabilities**: Advanced tweet and user search
- **Analytics Access**: Engagement and impression metrics
- **Media Handling**: Upload and manage images and videos
