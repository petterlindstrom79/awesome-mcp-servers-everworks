## Overview

Docker's official MCP registry for containerized MCP servers on Docker Hub.

## Features

- Search and browse catalog
- One-command install: `docker pull mcp/server`
- Security scanning, multi-arch support
- Ratings, download stats

## Use Cases

- Rapid MCP server deployment for agents
- Kubernetes/CI/CD integration
- Scalable agent tool ecosystems

## Comparisons

Docker-based MCP registry with pre-built images vs npm install for source code.

## Deployment

```bash
docker run -d mcp/server-name:latest
```

## Pricing

Free, open-source.
