## Overview

WireMCP is an MCP server for Wireshark (tshark) that provides LLMs with real-time network traffic analysis capabilities. It bridges tshark command-line tools with AI assistants for network diagnostics.

## Features

- Real-time network traffic analysis
- Integration with tshark/Wireshark
- Packet capture and analysis through MCP protocol
- Enables LLM-based network troubleshooting
