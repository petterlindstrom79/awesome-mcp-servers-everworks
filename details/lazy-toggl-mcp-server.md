## Overview

Lazy Toggl MCP Server is a lightweight and simplified implementation of a Toggl Track MCP server. It's designed for users who want quick setup and minimal configuration while still accessing essential time tracking features through AI assistants.

## Features

- **Minimal Configuration**: Quick and easy setup with minimal dependencies
- **Essential Time Tracking**: Core time tracking functionality without complexity
- **Start/Stop Timers**: Simple timer controls through natural language
- **Time Entry Queries**: Basic time entry listing and retrieval
- **Lightweight**: Minimal resource footprint
- **Fast Performance**: Optimized for quick responses
- **Simple API**: Straightforward integration with MCP clients

## Philosophy

The "lazy" approach prioritizes:
- Simplicity over feature completeness
- Quick setup over configuration options
- Essential functionality over comprehensive coverage
- Low maintenance over advanced capabilities

## Use Cases

- Personal time tracking for individual developers
- Quick time logging without complex setups
- Learning MCP server development with a simple example
- Minimal overhead time tracking for small teams
- Rapid prototyping of time tracking workflows

## Installation

Simple npm installation with minimal configuration required:

```bash
npm install lazy-toggl-mcp
```
