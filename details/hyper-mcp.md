## Overview

Hyper MCP is a high-performance, secure MCP server that uses WebAssembly plugins to extend its capabilities. The WASM-based plugin system enables safe execution of untrusted code while providing extensibility.

## Features

- WebAssembly-based plugin system for safe extensibility
- Fast and secure MCP server implementation
- Sandboxed plugin execution environment
- Extensible tool capabilities through WASM plugins
- Built for security-conscious deployments
