## Overview

A Model Context Protocol (MCP) server implementation that provides secure interaction with MySQL databases. Enables LLMs to inspect database schemas, execute SQL queries, and manage MySQL data through standardized MCP tools.

## Features

- Secure MySQL database connectivity
- Schema inspection capabilities
- SQL query execution
- Standardized MCP tool interface
- Supports common MySQL operations

## Usage

Configure with MySQL connection credentials including host, port, username, password, and database name. The server exposes MCP tools for database interaction.
