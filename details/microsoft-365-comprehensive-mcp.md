## Overview

Microsoft 365 Comprehensive MCP is an MCP server for Microsoft Office 365 Outlook that provides email, calendar, and SharePoint integration for Claude, ChatGPT, and AI assistants via Microsoft Graph API. It offers multi-account authentication and advanced search capabilities across the entire Microsoft 365 ecosystem.

## Features

- **Email Management**: Complete Outlook email operations
- **Calendar Operations**: Full calendar scheduling and management
- **OneDrive Integration**: File handling and document management
- **Contact Management**: Address book and contact operations
- **Multi-Account Support**: Manage multiple Microsoft 365 accounts
- **Advanced Search**: Powerful search across all services
- **SharePoint Integration**: Access SharePoint files and documents
- **Teams Integration**: Basic Microsoft Teams operations
