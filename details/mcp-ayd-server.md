## Overview

MCP Ayd Server is a Model Context Protocol server that integrates with Ayd, a status monitoring service. It allows LLM-based assistants to query the current status of services being monitored by Ayd.

## Features

- Query service status from Ayd monitoring platform
- Standardized MCP interface for AI assistants
- Real-time status information retrieval
- Supports multiple monitors through Ayd's API
