## Overview

Dive is an open-source MCP Host Desktop Application designed to connect Large Language Models with external tools through the Model Context Protocol. It provides a unified, user-friendly desktop interface for managing and interacting with MCP servers.

## Features

- Desktop application for MCP host management
- Integrates with any LLM supporting function calling
- Manages connections to multiple MCP servers
- Open-source and community-driven
- Unified interface for tool discovery and execution
- Cross-platform desktop experience for AI tool integration
