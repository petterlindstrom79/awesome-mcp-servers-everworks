## Overview

The Official DeepL Translator MCP Server combines the DeepL API with the Model Context Protocol to provide Claude and other MCP-compatible clients with access to DeepL's specialized translation capabilities. This official implementation ensures the highest quality translations and full feature support.

## Features

- **Text Translation**: Translate text between 30+ languages with optional automatic source detection
- **Target Language Control**: Specify target language and formality level (formal/informal)
- **Document Translation**: Support for PDF, DOCX, PPTX, XLSX, HTML, and TXT formats
- **Text Rephrasing**: Generate text variations with configurable writing style and tone
- **Glossary Management**: Access and utilize custom glossaries for consistent terminology
- **Glossary Listing**: Browse all available glossaries with metadata
- **Formality Control**: Adjust translation formality to match context and audience
- **Batch Processing**: Handle multiple translation requests efficiently

## Supported Document Formats

- PDF - Portable Document Format
- DOCX - Microsoft Word documents
- PPTX - Microsoft PowerPoint presentations
- XLSX - Microsoft Excel spreadsheets
- HTML - Web pages and HTML content
- TXT - Plain text files

## Use Cases

- Translate documentation and technical content with high accuracy
- Localize applications and websites through AI workflows
- Process multilingual customer support queries
- Create translations with consistent technical terminology using glossaries
- Generate professional document translations in multiple formats
- Rephrase content for different audiences and styles

## Integration

Requires a DeepL API key (free tier available). Can be installed directly from the GitHub repository or integrated into custom workflows.
