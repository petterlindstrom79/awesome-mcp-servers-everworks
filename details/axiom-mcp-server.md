## Overview

Axiom MCP Server provides Model Context Protocol integration for the Axiom observability platform. It enables AI assistants to perform complex log queries, event analysis, and data exploration within Axiom using natural language.

## Features

- Query Axiom datasets through MCP tools
- Analyze logs and traces with AI assistance
- Access Axiom's observability data programmatically
- Natural language log exploration
- Built by the Axiom team
