## Overview

YouTube Data API MCP provides real-time YouTube data access through the YouTube Data API v3. It enables AI assistants to search, analyze, and retrieve detailed information about YouTube videos, channels, playlists, and more through natural language conversations.

## Features

- **Video Search**: Search for videos by keywords, topics, and filters
- **Channel Management**: Access channel information and statistics
- **Playlist Operations**: Manage and query playlists
- **Shorts Integration**: Work with YouTube Shorts content
- **Advanced Analytics**: Retrieve detailed video and channel metrics
- **Comment Management**: Access and analyze video comments
- **Metadata Extraction**: Get comprehensive video metadata
- **Trending Videos**: Access trending content by region

## Video Operations

- Search videos by keyword
- Get video details (views, likes, duration)
- Access video statistics
- Retrieve video metadata
- List related videos
- Get video categories
- Access video thumbnails
- Query video captions availability

## Channel Features

- Get channel information
- Retrieve subscriber counts
- Access channel statistics
- List channel videos
- Get channel playlists
- Retrieve channel metadata
- Monitor channel activity

## Playlist Management

- List playlist items
- Get playlist details
- Access playlist statistics
- Retrieve playlist videos
- Search within playlists

## Analytics Capabilities

## API Requirements

Requires YouTube Data API v3 key (free with Google Cloud account, subject to quotas).

## Pricing

Free and open-source server. YouTube Data API has free tier with daily quota limits.
