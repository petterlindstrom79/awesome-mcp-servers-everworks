## Overview

OpenWeatherMap MCP Server is a comprehensive Model Context Protocol server that provides weather data and forecasts through the OpenWeatherMap API. It enables AI assistants to access real-time weather information, forecasts, air quality data, and location services.

## Features

- **Current Weather**: Real-time weather data for any location
- **5-Day Forecast**: Detailed weather predictions up to 5 days ahead
- **Air Quality Index**: Current air pollution levels and forecasts
- **Location Services**: Geocoding and reverse geocoding capabilities
- **Multi-location Support**: Query weather for multiple locations simultaneously
- **Unit Conversion**: Support for metric, imperial, and standard units
- **Weather Icons**: Access to weather condition icons and descriptions
- **Historical Data**: Access to historical weather information (with appropriate API tier)

## Weather Data Available

- Temperature (current, min, max, feels like)
- Atmospheric pressure
- Humidity percentage
- Weather conditions (clear, clouds, rain, snow, etc.)
- Wind speed and direction
- Cloudiness percentage
- Visibility distance
- Precipitation amounts
- Sunrise and sunset times

## Air Quality Features

- Air Quality Index (AQI)
- Pollutant concentrations (CO, NO, NO2, O3, SO2, PM2.5, PM10, NH3)
- Air quality forecasts
- Health recommendations based on AQI

## Use Cases

- Get weather updates through AI conversations
- Plan outdoor activities based on forecasts
- Monitor air quality for health considerations
- Integrate weather data into smart home automation
- Create location-based weather alerts
- Build weather-aware applications

## API Requirements

Requires OpenWeatherMap API key (free tier available with 1,000 calls/day).

## Pricing

Free and open-source server. OpenWeatherMap API offers free tier with generous limits.
