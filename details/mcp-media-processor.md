## Overview

MCP Media Processor is a Node.js server implementing Model Context Protocol (MCP) for media processing operations. It provides powerful video and image manipulation capabilities, integrating FFmpeg for video processing and ImageMagick for image operations.

## Features

- **Video Processing**: Convert, transcode, and manipulate video files
- **Video Compression**: Optimize video files with configurable quality settings
- **Image Processing**: Comprehensive image manipulation and editing
- **Image Compression**: Reduce image file sizes while maintaining quality
- **Format Conversion**: Support for numerous media formats
- **Batch Operations**: Process multiple files simultaneously
- **Metadata Extraction**: Read and modify media metadata
- **Thumbnail Generation**: Create video thumbnails and image previews

## Video Capabilities

- Video transcoding (MP4, WebM, AVI, MOV, etc.)
- Resolution and bitrate adjustment
- Frame rate conversion
- Audio extraction and manipulation
- Video trimming and cutting
- Watermark addition
- Format conversion
- Codec optimization

## Image Capabilities

- Format conversion (JPEG, PNG, WebP, GIF, etc.)
- Resizing and cropping
- Quality adjustment
- Compression optimization
- Filters and effects
- Color space conversion
- Batch image processing

## Requirements

- Node.js runtime
- FFmpeg for video processing operations
- ImageMagick for image processing operations

## Use Cases

- Optimize media for web delivery
- Create video transcoding pipelines
- Automate media processing workflows
- Generate thumbnails and previews
- Prepare media for different platforms
- Reduce storage costs through compression
