## Overview

Google Calendar Multi-Account MCP is an advanced Model Context Protocol server that provides comprehensive Google Calendar integration for AI assistants. It features multi-account support, cross-account conflict detection, smart scheduling with natural language understanding, and intelligent import capabilities.

## Features

- **Multi-Account Support**: Connect and manage multiple Google accounts simultaneously
- **Multi-Calendar Support**: Work with multiple calendars across accounts
- **Cross-Account Conflict Detection**: Automatically detect scheduling conflicts across all accounts
- **Event Management**: Create, update, delete, and search calendar events
- **Recurring Events**: Full support for recurring event patterns
- **Free/Busy Queries**: Check availability across calendars
- **Smart Scheduling**: Natural language event creation and scheduling
- **Intelligent Import**: Import events from images, PDFs, or web links
- **Time Zone Support**: Handle events across different time zones
- **Attendee Management**: Manage event attendees and RSVPs

## Smart Scheduling Features

- Natural language event parsing ("Schedule lunch with John next Tuesday at noon")
- Automatic time zone detection and conversion
- Smart conflict resolution suggestions
- Optimal meeting time recommendations
- Buffer time between meetings
- Travel time consideration

## Intelligent Import

- Extract event details from images (screenshots, photos of flyers)
- Parse PDFs for event information
- Import events from web links and emails
- Automatic date and time extraction
- Location parsing and geocoding

## Use Cases

- Manage personal and work calendars from a single interface
- Schedule meetings across multiple organizations
- Avoid double-booking across accounts
- Import events from various sources automatically
- Create events through natural conversation
- Coordinate schedules across teams

## API Operations

- List events with flexible filtering
- Create single and recurring events
- Update event details and attendees
- Delete events and event instances
- Search across all calendars
- Query free/busy information
- Manage calendar sharing and permissions

## Pricing

Free and open-source. Requires Google Calendar access (free with Google account).
