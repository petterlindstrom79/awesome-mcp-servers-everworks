## Overview

MCP server for MotherDuck and local DuckDB. Enables LLMs to interact with DuckDB's high-performance analytical database both locally and through MotherDuck's cloud service.

## Features

- Local DuckDB integration
- MotherDuck cloud connectivity
- SQL query execution
- Analytical query support
- Seamless data operations

## Pricing

DuckDB is open-source and free. MotherDuck offers a free tier with paid plans for additional resources.
