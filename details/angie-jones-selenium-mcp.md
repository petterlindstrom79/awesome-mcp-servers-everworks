## Overview

Angie Jones Selenium MCP is the official Selenium MCP server announced by Angie Jones, enabling browser automation through the Model Context Protocol for Selenium WebDriver. It brings industry-standard test automation capabilities to AI-powered workflows, making browser testing accessible through natural language.

## Features

- **WebDriver Integration**: Full Selenium WebDriver API access
- **Multi-Browser Support**: Chrome, Firefox, Safari, Edge
- **Element Location**: Multiple locator strategies
- **Action Chains**: Complex user interaction simulation
- **Wait Strategies**: Explicit and implicit waits
- **Screenshot Capture**: Visual verification and debugging
- **JavaScript Execution**: Execute custom JavaScript
- **Window Management**: Handle multiple windows and tabs
