## Overview

The HashiCorp Terraform MCP Server is the official Model Context Protocol integration for Terraform, providing AI models with real-time access to the Terraform ecosystem. This ensures AI-generated Terraform configurations use accurate, up-to-date information rather than potentially outdated training data.

## Features

- **Terraform Registry Integration**: Direct integration with public Terraform Registry APIs for providers, modules, and policies
- **HCP Terraform Support**: Full workspace management and organization/project listing capabilities
- **Terraform Enterprise**: Complete support for enterprise deployments with private registry access
- **Provider Documentation**: Real-time access to current Terraform provider documentation and schemas
- **Module Discovery**: Browse and access thousands of community and verified modules
- **Policy Integration**: Access Terraform policy documentation and best practices
- **Private Registry**: Connect to private Terraform registries for enterprise use cases

## Key Benefits

- Ensures AI assistants reference current Terraform documentation
- Reduces errors from outdated provider information in LLM training data
- Accelerates Infrastructure as Code development with accurate context
- Supports both public and private Terraform ecosystems
- Seamless integration with HashiCorp Cloud Platform

## Use Cases

- Generate accurate Terraform configurations with current provider syntax
- Discover and implement community modules efficiently
- Build infrastructure code with real-time documentation access
- Maintain consistency with enterprise Terraform standards
- Automate IaC workflows with AI-powered assistance
