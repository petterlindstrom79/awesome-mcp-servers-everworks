## Overview

Social Media Sync MCP is a universal Model Context Protocol server that provides seamless API access to 6 major social media platforms. It dynamically exposes 549 tools based on the API keys you provide, enabling comprehensive social media management through AI assistants.

## Supported Platforms

- **YouTube**: Video management and analytics
- **LinkedIn**: Professional networking and content
- **Facebook**: Social networking and pages
- **Instagram**: Photo and video sharing
- **TikTok**: Short-form video content
- **Twitter/X**: Microblogging and real-time updates

## Features

- **549 Dynamic Tools**: Tools exposed based on configured API keys
- **Multi-Platform Management**: Manage all platforms from one interface
- **Unified API**: Consistent interface across platforms
- **Batch Operations**: Perform actions across multiple platforms simultaneously
- **Analytics Access**: Retrieve metrics from all connected platforms
- **Content Publishing**: Post to multiple platforms at once
- **Engagement Tracking**: Monitor interactions across services
- **Scheduling Support**: Schedule posts across platforms

## Dynamic Tool Exposure

The server intelligently exposes tools based on:
- Configured API keys
- Platform permissions
- Account types (personal, business, creator)
- Available API endpoints

## Authentication

Supports multiple authentication methods:
- OAuth2 for modern platforms
- API key authentication
- Token-based access
- Automatic token refresh

## Pricing
