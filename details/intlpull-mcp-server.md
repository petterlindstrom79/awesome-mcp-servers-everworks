## Overview

IntlPull's MCP Server provides full translation management capabilities for Claude Desktop, Claude Code, and Cursor. It enables AI-powered internationalization workflows, making it easier for developers to manage translations across multiple languages and platforms.

## Features

- **Full Translation Management**: Complete lifecycle management for translation content
- **Multi-platform Support**: Works with Claude Desktop, Claude Code, and Cursor
- **Automated i18n Workflows**: Streamline translation processes with AI assistance
- **Project Synchronization**: Keep translations in sync across team members
- **Version Control Integration**: Track translation changes alongside code
- **Quality Assurance**: Automated checks for translation quality and consistency
- **Context Preservation**: Maintain context for accurate translations
- **Collaboration Tools**: Enable team collaboration on translations

## Integration

Seamlessly integrates with popular development environments:
- Claude Desktop for conversational translation management
- Claude Code for in-editor translation workflows
- Cursor for AI-powered development with i18n support

## Use Cases

- Manage application translations directly from your IDE
- Automate translation updates across multiple languages
- Collaborate with translators through AI-assisted workflows
- Maintain translation quality with automated checks
- Synchronize translations across development teams
