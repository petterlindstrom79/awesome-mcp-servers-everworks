## Overview

Image Compression MCP is a high-performance image compression microservice based on Modal Context Protocol. It features smart compression that automatically selects optimal compression parameters based on image content, batch processing for parallel compression of multiple images, and quality control with customizable compression quality to balance file size and visual quality.

## Features

- **Smart Compression**: AI-driven parameter selection for optimal results
- **Batch Processing**: Parallel compression of multiple images
- **Quality Control**: Customizable quality settings (1-100)
- **Format Support**: JPEG, PNG, WebP, and more
- **Size Optimization**: Automatic file size reduction
- **Visual Quality Preservation**: Maintain image quality during compression
- **Lossless Options**: Support for lossless compression when needed
- **Performance Metrics**: Track compression ratios and processing times

## Compression Features
