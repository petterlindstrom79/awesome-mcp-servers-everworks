# Google Gemini MCP Server

MCP server for Google Gemini supporting multimodal tasks across text, code, and images. Integrations produce design briefs, test plans, and visual analyses for rapid artifact-to-action conversion. Ideal for product and QA groups in fast feedback cycles.
