## Overview

Repository containing practical examples from official MCP tutorials for beginners.

## Examples

- **Quickstart MCP Weather Server**: Simple server demonstrating basic MCP functionality via weather data retrieval.
- **LLM-Powered Chatbot MCP Client**: Client example integrating MCP with an LLM chatbot.

## Security Considerations

Examples are minimal; for production network exposure (HTTP/SSE/WebSocket), implement authentication, CORS allowlist, request limits, timeouts, rate limits, and log redaction (see SECURITY.md).
