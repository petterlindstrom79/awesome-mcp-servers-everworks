## Overview

Evernote Notes MCP Server is a local Model Context Protocol server that connects Claude Desktop and other MCP-compatible LLMs with your Evernote account. It allows contextual queries and searches over notes using natural language, providing secure local AI-assisted access to your Evernote knowledge base.

## Features

- **Local Secure Access**: All data processing happens locally for privacy
- **Natural Language Search**: Query your notes using conversational language
- **Contextual Queries**: Ask questions about note content and get relevant results
- **Notebook Organization**: Search across notebooks and tags
- **Full-text Search**: Search within note content, titles, and tags
- **Note Retrieval**: Access complete note content through AI conversations
- **Tag-based Queries**: Filter notes by tags and categories
- **Date-based Search**: Find notes by creation or modification date

## Search Capabilities

- Search by keywords and phrases
- Query notes by topic or category
- Find notes with specific tags
- Locate notes in particular notebooks
- Search by date ranges
- Combine multiple search criteria
- Natural language query interpretation

## Use Cases

- Quickly find information buried in thousands of notes
- Summarize notes on specific topics
- Extract insights from research collections
- Organize and categorize notes through AI assistance
- Build knowledge graphs from note relationships
- Generate reports from note collections
- Rediscover forgotten notes and ideas

## Privacy and Security

- Runs entirely on your local machine
- No cloud processing of note content
- Direct connection to Evernote API
- Secure credential storage
- Full user control over data access

## Pricing

Free and open-source. Requires Evernote account (free or paid tiers).
