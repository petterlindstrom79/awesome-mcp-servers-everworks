## Overview

Envoy is an MCP server for TouchDesigner, the visual programming environment for real-time interactive multimedia content. It provides 45 tools that enable AI assistants to create operators, set parameters, wire connections, and manage TouchDesigner projects through natural language conversation.

## Features

- 45 TouchDesigner-specific MCP tools
- Create operators programmatically via AI
- Set and modify operator parameters
- Wire connections between operators
- Manage TouchDesigner projects
- Natural language interface for TouchDesigner operations
- Real-time interactive multimedia control

## Use Cases

- AI-assisted visual programming in TouchDesigner
- Automated creation of multimedia installations
- Conversational interface for creative coding workflows
- Prototyping interactive content with AI guidance
- Managing complex TouchDesigner project structures
